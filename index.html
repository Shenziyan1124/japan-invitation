展示会について

API Chinaは、中国の5000以上のAPI、化学医薬品、バイオ医薬品、製剤加工TCMメーカー、CRO/CMO/CDMOから6万人以上の製薬専門家を集めた展示会であり、国際的な製薬専門家にとって理想的なビジネスプラットフォームとなっています。

「中国製薬産業サイクルのグランドプラットフォーム、グローバル製薬産業チェーンの揺籃」をテーマとしており、原料、受託サービス、バイオ医薬品、技術、包装、機械などの企業が1200社以上毎回出展しています。

当展示会は約5,000社の中国製薬企業に最も好まれつつあるプラットフォームとなっており、展示会と調達に100強中の企業の97社が出展。
